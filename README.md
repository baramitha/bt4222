# BT4222 SLRA for Anime
Hybrid Anime Recommender: Matrix Factorization → NeuMF → Two Tower Network

## Setup
1. Clone: `git clone https://github.com/yourusername/BT4222-anime-recommender`
2. Colab: Click "Open in Colab" on any .ipynb file

## Structure
- `notebooks/`: Work in progress by steps
- `data/`: Kaggle MAL datasets
- `docs/`: Interim report & presentations

## Colab Workflow
1. Open notebook from GitHub repo
2. **File → Save a copy in GitHub** after changes
3. Use subfolder
